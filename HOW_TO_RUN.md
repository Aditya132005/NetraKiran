# Karan Optics — How to Run

## Quick Start (Two Terminals)

### Terminal 1 — Backend API
```bash
cd netra-kiran-optics/backend
npm run dev
# Starts on http://localhost:5000
```

### Terminal 2 — Frontend
```bash
cd netra-kiran-optics/frontend
npm run dev
# Opens on http://localhost:5173
```
